# Security automation artifacts

- **`securityheaders-rianell.com.md`** — Latest [SecurityHeaders.com](https://securityheaders.com/) scan for **rianell.com** (HTML → Markdown), updated by CI after GitHub Pages deploy on `main`/`master`.
- **`securityheaders-runs/run-*.md`** — One file per workflow run for history.

Generated by **`scripts/fetch-securityheaders-report.mjs`** (`.github/workflows/ci.yml` → job **Security Headers report**). Commits use `[skip ci]` to avoid workflow loops.

Secrets (`.env`, keys) stay gitignored per root **`.gitignore`**; these reports contain only public scan output.
